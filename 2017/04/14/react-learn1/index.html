React简介

  • 是一个用于构建用户界面的JavaScript库。
  • 起源于Facebook和Instagram的项目中。
  • 主要用于构建UI，很多人认为React是MVC中的V（视图）。
  • 拥有较高的性能，代码逻辑简单，便于使用。

React概述

React核心思想：封装组件，各个组件维护自己的状态和UI，当状态变更，自动重新渲染整个组件。
React核心内容：

  • 组件 页面当中的所有元素都是通过 React 组件来表达。
  • JSX JSX 自定义语法会转化为 JS 运行在页面中。
  • Virtual DOM 抽象出来的高性能的虚拟 DOM 树。
  • Data Flow 数据流向自顶向下的设计模式。
    React主要特点：
  • 高效 —最小化重绘DOM，React通过虚拟DOM，最大限度的减少与DOM的交互。基本原理是React把视图简化为一个render函数，render函数只关心两个参数，配置项和状态。render函数读入当前状态，并返回一个快速的DOM描述，只要React被告知状态有变化，其会重新运行函数，计算出新的DOM描述，然后判断差异后，进行最少的必要更新。涉及到的算法有虚拟DOM Diff算法。
  • 组件 —组件化开发思想，组件主要是封装起来具有独立功能的UI部件。将UI上每一个功能相对独立的模块定义成组件，然后将小的组件通过组合或嵌套的方式构建成大的组件，最终完成整个UI的构建。
  • 单一 —单事件处理器，web前端开发中，一般使用事件委托的方式减少多个事件处理，而React在整个应用中采用的是单个事件处理器，把所有事件委托放到这个事件处理器中。

React安装

安装：为了提高效率，一般会使用一些构建工具，常包括：

  • 包管理器 如npm
  • 打包工具 如Webpack
  • 编译工具 如Babel
    通过npm安装

然后从官网直接下载最新版本，会引入三个库： react.js 、react-dom.js 和 browser.min.js：

  • react.js - React 的核心库
  • react-dom.js - 提供与 DOM 相关的功能
  • browser.min.js - 用于将 JSX 语法转为 JavaScript 语法
